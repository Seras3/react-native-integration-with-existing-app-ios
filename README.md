# react-native-integration-with-existing-app-ios

1. Clone project
2. `yarn install`
3. cd ios/
4. `pod install`
5. cd .. 
6. `yarn start` 
7. (in other terminal tab) `npx react-native run-ios`

If 7. does not work, try to run app from Xcode => 
1. cd ios/  
2. open the **MyNativeApp.xcworkspace**
3. hit run
